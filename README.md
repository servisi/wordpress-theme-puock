## Kurulum

Temayı WordPress yönetim panelinden **Görünüm** > **Temalar** > **Yeni Ekle** bölümünden yükleyebilirsiniz.

**Önemli:** Tema kurulumundan önce verilerinizi yedeklemenizi öneririz.

### Sistem Gereksinimleri

- WordPress `6.0+`
- PHP `7.4+`
